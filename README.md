# Iceberry 🍦

## Objective

The primary objective of this project is to develop a functional and visually appealing website for **Iceberry**, an online dessert ordering platform. The website enables users to browse various ice cream and dessert options, add items to their cart, and place orders seamlessly. The project focuses on providing an attractive and engaging user experience while implementing essential web development technologies such as HTML, CSS, JavaScript, and jQuery.

## Project Description

This project involves the creation of a fully responsive and interactive website that showcases multiple dessert options, including ice creams, waffles, shakes, and sundaes. The website consists of multiple web pages with different functionalities to ensure a smooth ordering experience for customers.

## How the Website Works

### 1. Homepage

* Serves as the entry point of the website.
* Displays dessert categories with appealing visuals and animations.
* Provides essential information about the brand and its specialties.
* Technologies used: **HTML, CSS, jQuery**.

### 2. Menu Page

* Displays the complete list of desserts with images, descriptions, and prices.
* Allows users to browse and select their desired items.
* Technologies used: **HTML, CSS, jQuery**.

### 3. Cart Page

* Enables users to add items to the cart.
* Displays total price dynamically.
* Allows users to update quantities or remove items.
* Generates the bill for selected items.
* Technologies used: **HTML, CSS, JavaScript**.

### 4. Login Form Page

* Users are redirected to a login form before placing an order.
* Collects necessary customer details.
* Includes form validation to ensure correct input.
* Technologies used: **HTML, CSS, JavaScript**.

## Functionalities Used

### HTML (HyperText Markup Language)

* Structures web pages using elements like headings, paragraphs, images, forms, tables, lists, and divs.
* Uses semantic HTML for better accessibility and readability.

### CSS (Cascading Style Sheets)

* Styles the website with custom fonts, colors, spacing, and layouts.
* Uses **Flexbox** and **Grid** for responsive design.
* Adds hover effects and animations to improve user interaction.

### JavaScript

* Implements dynamic cart functionality such as adding and removing items.
* Updates total price in real-time.
* Handles form validation for order placement.

### jQuery

* Simplifies JavaScript for animations and menu toggling.
* Uses **AJAX** to improve user experience by avoiding full page reloads.

## Conclusion

This project successfully integrates essential web technologies to create a functional and aesthetically pleasing dessert ordering website. By combining structured HTML, styled CSS layouts, interactive JavaScript features, and enhanced jQuery effects, **Iceberry** provides a smooth and user‑friendly online dessert ordering experience.
